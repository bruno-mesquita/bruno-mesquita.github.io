# Bruno Mesquita Portfolio

Personal portfolio website built with Next.js, featuring a modern design with animations and internationalization support.

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS + shadcn/ui
- **Package Manager:** Bun
- **Deployment:** GitHub Pages (static export)

## Features

- 🌐 Internationalization (English / Portuguese)
- 🎨 Modern dark/light theme
- ✨ Smooth animations
- 📱 Fully responsive
- 🔗 Social links integration

## Getting Started

```bash
# Install dependencies
bun install

# Run development server
bun run dev

# Build for production
bun run build
```

## Project Structure

```
app/           → Next.js App Router pages
components/    → React components (portfolio + shadcn/ui)
data/          → Static data (projects, skills)
lib/           → Utilities
public/        → Static assets
```
